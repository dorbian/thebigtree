Forest Dashboard

Welcome to the forest control room. This is a quick start and reference for the bot and site.

Quick Start

1) Use /auth in Discord to get a 24h web token if your role allows it.

2) Paste the token on this page to unlock the tools.

3) Pick a panel on the left to open the manager you need.

Discord Bot Commands

Bingo: /bingo, /bingo-menu, /bingo-stage, /call, /bingo-status, /bingo-end, /bingo-url, /bingo-owner-url, /buycard, /buycards, /bingo-upload-bg.

Tarot: /tarot_start, /tarot_draw, /tarot_flip, /tarot_end, /tarot_stream, /tarot_add_card, /tarot_addcard, /tarot_session, /tarot_template_init, /tarot_deck_from_template, /tarot_claims_post.

Contests: /contest to start, /contest_stop to end.

Permissions: /permissions list, add-role, remove-role, add-user, remove-user, refresh.

Announcements: /commune to craft an official post.

AI helper: /quickpost for a short draft with approve/retry.

Site & Overlay Features

Dashboard: home base, quick reference, and credits.

Session Manager: create and run tarot sessions, open player and priestess views.

Deck Editor: build decks and cards, edit meanings, suits, numbers, and art.

Bingo Manager: manage live games, claims, and stage changes.

Overlay Mode: OBS-friendly layouts for stream scenes.

Admin Tools: auth roles, auth keys, media library, artist index, gallery calendar.

Tarot Workflow

1) Create a session with a deck and spread.

2) Use Draw to pull cards and Reveal to flip them.

3) Click a card to reveal or fullscreen it.

4) Finish closes the session and enables screenshots.

Deck Editor Workflow

1) Pick a deck to load, then select a card or create a new one.

2) Set number, suit, meanings, flavor text, and art.

3) Theme weights apply only to the selected suit.

4) Use Edit Deck to change name, theme, and back art.

Troubleshooting

If the overlay is blank, check your auth token.

If cards do not update, refresh the reading view.

Bad gateway means the main server is still starting.

If images fail, reopen the media library.
